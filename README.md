# Event Manager App

A simple web-based event management system where users can:
- Create events
- Delete events
- Manage multiple timers

## Tech Used
- HTML
- CSS
- JavaScript

## Features
- Add Event
- Delete Event
- Delete All Events

## How to Run
1. Clone repo
2. Open in browser / run project
